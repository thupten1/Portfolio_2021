Website van Noek

WAT IS MIJN BIJDRAGE?

Een van de deliverable voor Noek was een website. Jeroen heeft de schets gemaakt voor de website, ik heb de website uitgewerkt. Jeroen zijn schets heb ik gecombineerd met de resultaat van Task analysis en Card sorting. (lees Task analysis en Card sorting voor meer informatie). Tevens heb ik cardsorting niet helemaal kunnen toepassen omdat ik gekozen had voor een one page website. De achtergrond is een fade van 2 kleuren het gaat van donker blauw naar zwart. Dit geeft een mooie effect en lijkt alsof je dieper het water in gaat. Op de website staat de bio, shows, video, muziek en contact gegevens. Momenteel hebben ik als test, muziekvideo van Eefje de visser gebruikt dit heb ik gedaan omdat Noek nog geen muziekvideo heeft.

WAT HEB IK HIERVAN GELEERD?

Ik heb geleerd dat ik eerst moet onderzoeken wat het beste is voor de website. Bijvoorbeeld eerst onderzoeken wat mensen echt willen zien op de website, waarvoor mensen op de website komen. Hierdoor kan ik de website veel beter indelen. Verder heb ik ook geleerd om te denken over de structuur van de website. Ik heb ook nog kennis opgedaan in coderen met name Bootstrap.
